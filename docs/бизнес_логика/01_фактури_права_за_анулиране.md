## Текуща Имполентация

**Файл:** `backend/graphql/mutations.py:3445-3447`


## Промяна 1: Разширяване на Правата

### Преди

```python
if current_status == 'paid' and new_status == 'cancelled' and current_user.role.name != "super_admin":
```

### След

```python

```

### Нов Текст на Съобщението

```python
raise PermissionDeniedException(
    detail="Нямате права да анулирате платена фактура. За това действие са необходими права на счетоводител или по-високи."
)
```

## Промяна 2: Логване за Одит

### Къде

След успешното анулиране на платена фактура, преди commit.

### Код

```python
# Log cancellation of paid invoice for audit
from backend.database.models import OperationLog

operation_log = OperationLog(
    operation_type="CANCEL_PAID_INVOICE",
    entity_type="Invoice",
    entity_id=invoice.id,
    user_id=current_user.id,
    company_id=current_user.company_id,
    details={
        "invoice_number": invoice.number,
        "invoice_id": invoice.id,
        "previous_status": current_status,
        "new_status": "cancelled",
        "amount": float(invoice.total),
        "vat_amount": float(invoice.vat_amount),
        "role": current_user.role.name,
        "user_email": current_user.email
    }
)
db.add(operation_log)
```
