## 3. ЛОГИКА ЗА ИЗЧИСЛЯВАНЕ

### Пример: Торта с 10 парчета, 5 съставки по 300g (1500g общо)

```
ПРОМЯНА: 10 → 24 парчета
─────────────────────────────────────────────

Стъпка 1: Изчисли количество за 1 парче
  300g / 10 = 30g за 1 парче

Стъпка 2: Умножи по новия брой парчета
  30g × 24 = 720g за новия брой парчета

Резултат:
  Всяка съставка: 300g → 480g
  Общо тегло: 1500g → 3600g
```

### Текуща имплементация
```tsx
const handlePiecesChange = (newPieces: number) => {
  if (newPieces <= 0) return;
  const oldPieces = recipeForm.currentPieces;
  const ratio = newPieces / oldPieces;
  
  const updatedSections = recipeForm.sections.map((section) => ({
    ...section,
    ingredients: section.ingredients.map((ing) => ({
      ...ing,
      quantityGross: String(Number(ing.quantityGross || '0') * ratio)
    }))
  }));
  
  setRecipeForm({
    ...recipeForm,
    currentPieces: newPieces,
    defaultPieces: newPieces,
    sections: updatedSections
  });
};
```
